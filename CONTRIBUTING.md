# 贡献指南

感谢你对 B站数据看板 项目的关注！

## 🚀 如何贡献

### 报告问题

如果你发现了bug或有功能建议，请通过 [Issue](https://github.com/yourusername/bilibili-dashboard/issues) 提交。

提交Issue时请包含：
- 问题的详细描述
- 复现步骤
- 期望的行为
- 截图（如果有）
- 运行环境（Windows版本、Python版本等）

### 提交代码

1. **Fork** 本仓库
2. 创建你的功能分支：`git checkout -b feature/AmazingFeature`
3. 提交你的修改：`git commit -m 'Add some AmazingFeature'`
4. 推送到分支：`git push origin feature/AmazingFeature`
5. 打开一个 **Pull Request**

### 代码规范

- 遵循 PEP 8 Python代码规范
- 添加必要的注释和文档字符串
- 保持代码简洁易读
- 测试你的修改确保不会破坏现有功能

### 开发环境搭建

```bash
# 克隆仓库
git clone https://github.com/yourusername/bilibili-dashboard.git
cd bilibili-dashboard

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/bilibili_dashboard.py
```
